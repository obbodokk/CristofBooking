# 🏨 CristofBooking  
### *Веб-приложение для бронирования отелей*

> Современная платформа для поиска и бронирования отелей с удобным интерфейсом и системой оплаты.

## ⚙️ Функционал приложения

- ✅ **Регистрация пользователей**  
  - создание аккаунта

- 🏨 **Просмотр доступных отелей**  
  - карточки отелей с фото, описанием и ценой

- 💳 **Оплата бронирования**  
  - Будет фейкоплата для примера
  - Возможность оформления рассрочки
  
- 🔍 **Фильтрация и поиск**  
  - По **рейтингу** отеля  
  - По **локации** (город, район)  
  - По **цене** за ночь
